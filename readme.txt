Wordpress plugin for Importing your Buzzsprout's podcast and play on your website.

== Description ==
This plugin creates a podcast page where you can import all your podcasts hosted on Buzzsprout through Buzzsprout feed URL. It also fetches description and transcript of the podcast. 

= Buzzsprout Service =
[Buzzsprout](https://www.buzzsprout.com) is a podcast hosting service. This plugin helps buzzsprout's users to import their podcasts into their WordPress website. We are not using or storing any private details of the user's buzzsprout account. We only use the RSS feed of the user's buzzsprout account which is publicly available to use. 
Note, this plugin is dependent on RSS feed, we import all the hosted podcasts using XML data provided by the RSS feed URL of buzzsprout account.

== Installation ==
1. After activation of this plugin you will find setting page inside podcast menu.
2. Enter buzzsprout rss feed and hit the save button (You can find this feed url at Directories > RSS feed in your Buzzsprout dashboard.)
3. Now you can see all your podcast list at Add and Update podcast page.
